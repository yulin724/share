# Share - AI 文档资源库

这是一个 AI 相关文档和资源的分享仓库，包含 AI 领域的中文翻译文档和导航页面。

## 📁 目录结构

```
share/
├── ai_docs/                    # AI 文档目录
│   ├── navi.html              # AI 文档导航页面
│   ├── AiInTheEnterprise中文翻译.pdf
│   ├── APracticalGuideToBuildingAgents中文翻译.pdf
│   └── PromptEngineering中文翻译.pdf
└── README.md                   # 项目说明文档
```

## 🚀 快速开始

### 访问导航页面

打开 `ai_docs/navi.html` 文件即可访问 AI 文档导航页面，该页面提供了：

- 现代化的 UI 设计
- 响应式布局，支持移动端访问
- 快速访问 AI 相关资源链接

### 本地查看

1. 克隆仓库：
```bash
git clone git@github.com:yulin724/share.git
cd share
```

2. 在浏览器中打开导航页面：
```bash
open ai_docs/navi.html
# 或使用其他浏览器打开
```

## 📚 文档说明

### PDF 文档

- **AiInTheEnterprise中文翻译.pdf** - AI 在企业中的应用（中文翻译版）
